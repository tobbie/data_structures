#  Javascript solutions to common data structures and algorithm problems

### Data Structures covered include:

	* Arrays
	* LinkedLists
	* Strings
	* Graphs 
	* Stacks
	* Queues
	* Trees - Binary Search Trees and Binary Trees

### Algorithms covered include:
	* Sorting
	* Searching
	* Recursion
